# minesweeper
Minesweeper written in C using SDL

- [x] Save, load function
- [x] Customizable
- [x] Time limit

# usage

`minesweeper [Row number] [Column number] [Mine count] [Time limit]`

Save with **F1**
Load with in menu or in game with **F2**
